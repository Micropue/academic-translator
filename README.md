# 学术翻译器
<img src="https://img.shields.io/badge/build-electron-blue?style=for-the-badge" /> <img src="https://img.shields.io/badge/vibe_coding-deepseek_v4_pro-green?style=for-the-badge" />

这是一个非常有意思的软件，它会把非常简单的白话文翻译成非常“烦人”的各种学术词汇掺杂的一句话。

<img width="500" alt="image" src="https://github.com/user-attachments/assets/05f85cb1-c62c-4ec3-89b4-ecb1cb8ce7ac" />


比如说：
```
小明喜欢喝奶茶
```
翻译结果：
```
碳基生物体小明对含乳制糖分溶液（即奶茶类饮品）产生显著的正向情感倾向，并频繁实施摄入行为
```

 - 它会翻译的非常专业，对于长句子也是支持的。
 - 我觉得部分时候可以用来给论文扩充字数。
 - 有的时候你还可以恶搞朋友一下
